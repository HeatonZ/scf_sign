## 创建空白函数

在[腾讯云函数](https://console.cloud.tencent.com/scf/index/1)中，新建建**py2.7**的空白函数，创建完毕后 函数配置-超时时间-10秒

## 签到函数
复制ele.py中的内容，填入函数代码，点击保存。

#### cookie
```
********* 替换为 ele的cookie中的SID
&&&&&&&&& 替换为 ele的cookie中的USERID
```

#### 以下这些功能不需要可以自行注释掉  

## 定时触发
进入创建好的函数，触发方式，添加并选择适合自己的触发方式  

例如：  
名称：`自行决定`  
触发周期：`自定义触发周期`  
Cron表达式：`0 0 9 * * * *`（每天九点执行）
